# Asteroids in Rust

Simple asteroids arcade game made using rust
and raylib. To learn rust.

Controls
* `A` - rotate left
* `W` - move forward
* `D` - rotate right
* `SPACE` - shoot

## Build

Rust version `1.83.0`
Cargo version `1.83.0`

```
git clone https://github.com/KeanBuyst/Asteroids-rs.git
cd ./Asteroids-rs
cargo run
```
